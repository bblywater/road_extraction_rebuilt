# 架构图表述说明

## 1. 基线 U-Net 总体结构图

- 图题：基线 U-Net 编码器-解码器结构示意图
- 图注：基线 U-Net 由四级编码器、一个瓶颈层和四级解码器构成。编码器采用双卷积块与最大池化逐级提取深层语义特征，解码器采用双线性上采样并与对应跳跃连接特征拼接，从而逐步恢复空间分辨率并输出道路分割结果。
- 图中元素：
  - 输入：`3 x 512 x 512`
  - 编码器四级特征：`32, 64, 128, 256`
  - 瓶颈层特征：`512`
  - 解码器四级恢复：`256, 128, 64, 32`
  - 每一级模块标注为 `Conv 3x3 + BN + ReLU` 两次
  - 跳跃连接用横向箭头表示
  - 输出层标注为 `1 x 1 Conv -> Logits`

## 2. DLAM 模块结构图

- 图题：轻量级双线性注意力模块 DLAM 结构示意图
- 图注：DLAM 由通道注意力分支和空间注意力分支组成。通道分支通过全局平均池化和全局最大池化聚合特征，再经共享多层感知机生成通道权重；空间分支通过水平条形卷积与垂直条形卷积建模道路的方向性上下文，再生成空间权重图。两个权重与输入特征逐点相乘后通过残差方式回加到原始特征。
- 图中元素：
  - 输入特征：`F`
  - 通道分支：`AvgPool / MaxPool -> Shared MLP -> Sigmoid -> Ac(F)`
  - 空间分支：`Conv 1xk + Conv kx1 -> Conv 1x1 -> Sigmoid -> As(F)`
  - 融合公式单独标出：`F' = F + F * Ac(F) * As(F)`
  - 输出特征：`F'`

## 3. DLGU-Net 集成结构图

- 图题：DLGU-Net 中 DLAM 模块的跳跃连接集成方式
- 图注：DLGU-Net 保持基线 U-Net 的主干拓扑不变，仅在四条跳跃连接进入解码器之前插入 DLAM 模块，对浅层特征进行通道与空间双重重标定，再将增强特征送入对应解码层。
- 图中元素：
  - 左侧为四级编码器，右侧为四级解码器
  - 四条 skip path 中间分别插入一个 `DLAM`
  - 在图中突出标注“主干不变，skip 增强”
  - 输出层保持与 baseline 一致

## 4. 空洞卷积增强基线结构图

- 图题：空洞卷积增强基线中的瓶颈层上下文模块示意图
- 图注：在保持 baseline 编码器与解码器结构不变的前提下，本文在瓶颈层引入四个并行空洞卷积分支，膨胀率分别为 1、2、4、8。各分支特征在通道维度拼接后经 `1 x 1` 卷积融合，并与输入瓶颈特征形成残差连接，用于增强多尺度道路上下文表达。
- 图中元素：
  - 输入瓶颈特征：`F`
  - 四个并行分支：`Dilated Conv r=1`、`r=2`、`r=4`、`r=8`
  - 拼接节点：`Concat`
  - 融合节点：`1 x 1 Conv`
  - 残差相加节点：`Fctx = F + phi([D1(F), D2(F), D4(F), D8(F)])`
  - 输出连接回解码器
